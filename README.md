# Sala de Estudio

Proyecto desarrollado para el curso Fundamentos de Ingeniería de Software.

## Equipo

- José Ignacio Leiva
- Vicente Araya
- Matías Henríquez
- Martín Leon

## Descripción

Esta aplicación está diseñada para organizar y visualizar fácilmente los bloques horarios disponibles en las salas de estudio de la universidad. Permite a los usuarios reservar y cancelar reservas de salas de estudio, así como consultar qué horarios ya están ocupados — todo en un solo lugar, reemplazando el proceso actual que se gestiona completamente por correo electrónico.
